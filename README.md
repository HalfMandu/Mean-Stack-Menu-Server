
## General info
Node backend for menu application

## Important commands
    npm install
    npm start
	
## Example
	GET https://localhost:3443/dishes?featured=true	
	POST https://localhost:3443/users/login 

## Uses
* Node v16.13.0
* Express
* Passport
* Mongoose
* Jade
* Cors
* Cookie-parser
* Jsonwebtoken
* Transitions
* Session-file-store

## Serves
* Dishes
* Favorites
* Leaders
* Promotions
* Users
